这是一个二级目录

root/test/22.jpg

root/test.sh

ssh：连接服务器

pwd：查看当前位置

ifconfig：查看网络信息

free：查看内存 -h

这是一个三目录 - 1

df -lh：查看磁盘

mkdir test：新建目录

rmdir：删除目录

cd：切换目录

git clone [http://]：git拉去代码到项目目录

ls：列出目录文件

du -sh *：查看目录占用

cat 1.txt：输入文件内容

uanme -a：查看系统版本

这是一个三目录 - 2

yum：CentOS

apt-get：Ubuntu

opkg：OpenWrt

这是一个三目录 - 3

which ifconfig：查看软件安装位置

wget [http://]：下载链接内容

tar -zxvf [文件]：解压文件

find -name “.jpg“：查找所有.jpg的文件

cp ./test/22.jpg ./：把22.jpg复制到上一层

mv 22.jpg 33.jpg：把22改成33

nohub a.sh：后台启动a.sh脚本

jobs：查看后台任务

ps：查看进程

ps -ef|grep “java”：通过管道符 配合grep命令 只筛选出java进程

netstat -ntlp：查看端口

curl [链接]：访问链接或ip

tail -n 10 test.txt：查看test.sh最新的10行

sz test.sh：把test.sh下载到本地

vim test.sh：文本编辑器

kill -9 %1：杀死进程 不确定杀的啥

top：查看进程

chmod a+x test.sh：给test.sh添加可执行权限

history：查看历史命令
